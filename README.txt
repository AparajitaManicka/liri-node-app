Liri-Node-App:
This is the node.js homework assignment for the UCF Coding Bootcamp.

In this assignment, we were asked to create a Liri Node app using Node.js.

Getting Started
Liri

Instructions:

This Liri app is designed with 4 options as below.
1.Movie-this
2.My-tweets
3.Spotify-this-song
4.Do-what-it-says

Movie this hits the omdb api to get the information of the movie specified. If no movie is specifies it is defaulted to the movie Mr.Nobody.
My-tweets hits the twitter api and retrieves recent 20 tweets.
Spotify-this-song hits the spotify api to get the information of the song specified. If no song is specified it it defaulted to The Sign from the band Ace of Base.
Do-what-it-says reads from the random.txt file and executes as per the instruction in that file.
